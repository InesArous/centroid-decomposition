This folder contain the following files:
	* recovery.py: python code to recover the missing values
	* monetdb_udf.sql: the sql file to use it with Monetdb
	* sample.txt: original data containing missing values (The null values in the first column are considered as missing values)
	* sample_recovered.txt: The recovered data
	* input_data.png: Figure of the input data
	* output_data.png: Figure of the output data
